# Java Course FMI Solutions

This repository contains my solutions to laboratory tasks and homeworks from the **Modern Java Technologies** course at the **Faculty of Mathematics and Informatics, Sofia University**.

The goal of this repository is to document my progress through the course while building a well-structured Java codebase and maintaining consistent GitHub activity.

---

## Topics Covered

The course explores a variety of Java and backend-related concepts including:

- Java fundamentals
- Object-Oriented Programming
- Collections and Clean Code
- Generics
- Unit Testing and Mocking
- I/O Streams and Files
- Lambdas and Stream API
- Multithreading
- Network Programming
- HTTP, REST and JSON
- Design Patterns
